日常を離れ、自分だけの時間を愉しむ。 そんな「ひとつ上の滞在」を、熱海で。

山龢には、貴賓館、本館あわせて
8つの客室があります。
それぞれに趣の異なる、
個性豊かな空間です。

扉を開ければ、やわらかな風とともに、
自然の気配がそっと入り込みます。

モダンでハイクラスなインテリアと、
情緒豊かな景色が重なり合い、
ここで過ごす時間に、
静かな奥行きをもたらします。

( Omotenashi )

山龢のおもてなし

( 01 ) 全室に、天然温泉の露天風呂を。

すべての客室に、露天風呂がついています。体の修復を助ける、やわらかな硫酸塩温泉が心と身体を癒します。

( 02 ) 創業１９０年の最高級寝具を。

すべてのお部屋の寝具は、創業1830年、京都の老舗・イワタ社の最高級品を使用しています。

( 03 ) 静寂に包まれる滞在を。

客室露天風呂や個室でのお食事など、プライベートな時間を大切にした設え。周囲を気にせず、心ゆくまでお寛ぎいただけます。

( Floor Map )
